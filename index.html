El Darwin Core (DwC, https://dwc.tdwg.org/)

  • Estándar de datos mantenido por la comunidad internacional de informática de la biodiversidad
  • Incluye un glosario de términos con identificadores, etiquetas y definiciones que facilitan el intercambio de información sobre biodiversidad.
  • Permite cuatro tipos de datos principales:
    • Metadatos
    • Registros de presencia
    • Listas de especies
    • Datos de eventos de muestreo

Mapeo de datos a los términos del DwC

El Darwin Core Archive (DwC-A)

    Archivo ZIP que contiene:

  • Archivo de texto con datos del núcleo
  • Archivo XML con descriptores de los datos
  • Archivo XML de metadatos según el estándar EML
  • Archivos de texto opcionales con datos de extensiones (ej. nombres comunes, imágenes, descripciones de especies)

Áreas de distribución de especies

  • Polígonos correspondientes al área que habita una especie
  • Se distribuyen en formatos geoespaciales (ej. shapefiles)

QGIS (https://qgis.org/)

  • Sistema de información geográfica (SIG) de código libre y abierto
  • Multiplataforma (Linux, Mac OS, Windows, ...)
  • Disponible en varios idiomas
  • Programable y extendible a través de complementos (plugins)

Complementos de QGIS (https://plugins.qgis.org/)

  • Aplicaciones desarrolladas independientemente, pero que se ejecutan en el ambiente de QGIS
  • Extienden y personalizan la funcionalidad principal de QGIS
  • La mayoría están programados en el lenguaje Python

PostgreSQL (https://www.postgresql.org/) y PostGIS (https://postgis.net/)

  • PostgreSQL es un motor de bases de datos de código libre y abierto
  • PostGIS incorpora en PostgreSQL tipos de datos geoespaciales (puntos, líneas, polígonos, raster) y funciones de geoprocesamiento
